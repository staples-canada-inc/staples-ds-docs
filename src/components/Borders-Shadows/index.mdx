# Lines, Borders & Shadows

All borders, horizontal and vertical dividers are `light-gray` with a stroke of 1px. All element outlines have a corner radius value of zero unless noted otherwise.


## Border

Use border utilities to add or remove an element's borders. Choose class `.border` for all borders or specify a specific side as noted below.

## Lines

### Vertical Lines

Use line utilities to quickly style lines. This can be done by using border-left or border-right and adjusting its length, and size.


<ComponentPreview>
    <div class="p-5 m-3 d-inline-flex border-left"></div>
</ComponentPreview>

```html
<div class="border-left"></div>
```

### Horizontal Line

Create horizontal lines by using the `<hr>` tag. 

<ComponentPreview>
    <hr/>
</ComponentPreview>

```html
<hr />
```


## Drop Shadows
Add or remove shadows to elements with box-shadow utilities.

You can quickly add or remove a shadow with our `box-shadow` utility classes. Includes support for `.shadow-none` and three default sizes.

<ComponentPreview>
<div class="d-flex flex-wrap justify-content-between">
    <div class="shadow-none p-5 m-3 bg-light border">No shadow</div>
    <div class="shadow-sm p-5 m-3 bg-light border">Small shadow</div>
    <div class="shadow p-5 m-3 bg-light border">Regular shadow</div>
    <div class="shadow-lg p-5 m-3 bg-light border">Larger shadow</div>            
</div>
</ComponentPreview>


<CodePreview>
