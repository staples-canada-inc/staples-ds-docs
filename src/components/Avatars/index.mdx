# Avatars

Avatars are typically profile images that either contain a placeholder, initials, or a custom image that the user can upload.

## Avatar with Image

Avatars with an image use the `<img>` tag and include the `.avatar-img` class. This class uses css to manipulate an image into an avatar. It can also be used for avatar placeholder icons.

## Avatar With User Initials

For creating avatars with the user's initials, use the `.avatar` class which creates the size of the image and background. Before inputting the initials, use the `span` tag and then use our custom `initials` class to properly apply the initials in the avatar. 

Once again, We also have different sizes available for these avatars {sm|lg|xl} in addition to the default size (MD in this case).

## Usage

Avatars are usually used as profile icons and in toasters when getting notifications. They can be customized to the user and serve as identification.
