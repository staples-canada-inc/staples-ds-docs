# Typography

Typography works to establish clear hierarchy, organize information and guide users through the content of a product or experience.

### Official Typeface

**TT Norms** is the official digital typeface of Staples. Lulo Clean is to never be used for digital as it does not meet online accessibility standards.

<button class="btn btn-secondary">[Download TT Norms](#)</button>

### Weights

Font weight is an essential typographic variable that can add emphasis and differentiate content to establish hierarchy. Font weight and size pairings must be carefully balanced. A bold weight will have more emphasis than a lighter weight of the same font size. However, a lighter weight can have more prominence than its bold counterpart if larger in size.

We suggest using **TT Norms Regular, Medium and Bold** for Staples digital.

## Usage

Type in the Staples Design System is managed using type tokens. This allows us to remain consistent across all our digital interfaces and applications.

### Type Tokens

Staples uses type tokens to manage typography. Type tokens are pre-set configurations of typographic elements such as font size, weight, or leading (line height). Selecting the appropriate type style is determined by layout and template structure. Layouts may have several levels of architecture or areas that require varying typographic hierarchies.  Refer to the typography guidelines and best practices when pairing header and paragraph styles together to establish visual hierarchy.

### Font Family

```scss

$font-family-regular: "TT Norms Regular";
$font-family-medium: "TT Norms Medium";
$font-family-bold: "TT Norms Bold";

```

### Sans-serif Font Stack

```scss

$font-family-sans-serif: $font-family-regular !default;

```

### Mono Font Stack

```scss

$font-family-monospace:
    SFMono-Regular,
    Menlo,
    Monaco,
    Consolas,
    "Liberation Mono",
    "Courier New",
    monospace !default;

```
