# Lists

Lists consist of related content grouped together and organized vertically.

## Unordered & Ordered List

Use the `<o>` tag to create an unordered list that isn't numbered.

## Usage

### Variations

| List Type      | Purpose                                                                    |
| :------------- | :------------------------------------------------------------------------- |
| Unordered List | To present content of equal status or value.                               |
| Ordered List   | Imply sequence and order. Commonly used when giving a set of instructions. |
| List Group     | To present content of equal status or value.                               |

### Length

Typically, lists should be used to present simple or short pieces of information.

### Order

Arrange list items in a logical way. For example, if the list pertains to resource use, the default order might be highest resource use to lowest. Whereas, grouping items into smaller sub-categories might be more meaningful in other contexts. Alternatively, alphabetical or numerical order is also widely used.

### Text

Ensure list items are grammatically parallel. For example, do not mix passive voice with active voice or declarative sentences (statements) with imperative sentences (direct command).

### Style

The `list-style-type` property specifies the type of list item marker.

```css
ul.a {
  list-style-type: circle;
}

ul.b {
  list-style-type: square;
}

ol.c {
  list-style-type: upper-roman;
}

ol.d {
  list-style-type: lower-alpha;
}
```

The `list-style-image` property specifies an image as the list item marker.

```css
ul {
  list-style-image: url('sqpurple.gif');
}
```

The `list-style-position` property specifies the position of the list-item markers (bullet points).

```css
ul.a {
  list-style-position: outside;
}

ul.b {
  list-style-position: inside;
}
```
