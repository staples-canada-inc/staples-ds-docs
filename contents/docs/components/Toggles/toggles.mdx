# Toggle

A toggle or switch has the markup of a custom checkbox but uses the `.custom-switch` class to render a toggle switch. Toggles also support the `disabled` attribute.

### Simple Toggle

<ComponentPreview>
	<FormGroup>
		<CustomInput
			type="switch"
			id="exampleCustomSwitch"
			name="customSwitch"
			label="Toggle this switch element"
		/>
		<CustomInput
			type="switch"
			id="exampleCustomSwitch2"
			name="customSwitch"
			label="Or this one!"
		/>
	</FormGroup>
</ComponentPreview>

```  html
    <div class="custom-control custom-switch">
        <input type="checkboxw" class="custom-control-input" name="customSwitch" id="customSwitch1">
        <label class="custom-control-label" for="customSwitch1">Toggle this switch element</label>
    </div>
    <div class="custom-control custom-switch">
        <input type="checkbox" class="custom-control-input" name="customSwitch" id="customSwitch2">
        <label class="custom-control-label" for="customSwitch2">Or this one!</label>
    </div>
```

### Disabled Toggle

	<FormGroup>
