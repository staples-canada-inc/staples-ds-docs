## Fluid Images
Use the `fluid` to scale image nicely to the parent element.

<ComponentPreview>
    <Image src="https://source.unsplash.com/1500x250?ocean" fluid />
</ComponentPreview>

<CodePreview>
<HtmlCodeBlock>

```html
<img src="..." alt="..." class="img-fluid" style="width: 100%; display: block;" />
```

</HtmlCodeBlock>
<ReactCodeBlock>

```html
<Image src="..." alt="..." fluid />
```

</ReactCodeBlock>
</CodePreview>

## Figures
Anytime you need to display a piece of content, like an image with an optional caption, consider using a `Figure`.

<ComponentPreview>
    <Figure>
        <Figure.Image
            width={650}
            height={250}
            alt="650x250"
            src="https://source.unsplash.com/650x250?ocean"
            fluid />
        <Figure.Caption>
            Nulla vitae elit libero, a pharetra augue mollis interdum.
        </Figure.Caption>
    </Figure>
</ComponentPreview>

<CodePreview>
<HtmlCodeBlock>

```html
<figure class="figure">
    <img width="171" height="180" alt="..." src="..." class="figure-img img-fluid" />
    <figcaption class="figure-caption">Nulla vitae elit libero, a pharetra augue mollis interdum.</figcaption>
</figure>
```

</HtmlCodeBlock>
<ReactCodeBlock>

```html
<Figure>
    <Figure.Image
        width={650}
        height={250}
        alt="650x250"
        src="https://source.unsplash.com/650x250?ocean"
        fluid />
    <Figure.Caption>
        Nulla vitae elit libero, a pharetra augue mollis interdum.
    </Figure.Caption>
</Figure>
```

</ReactCodeBlock>
</CodePreview>
