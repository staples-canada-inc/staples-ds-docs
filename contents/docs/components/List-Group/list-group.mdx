# List Group

## Basic Example

A list group is an unordered list with list items and proper classes. Build upon it with the options that follow, or with your own CSS as needed.

<ComponentPreview>
  <ListGroup>
    <ListGroup.Item>Cras justo odio</ListGroup.Item>
    <ListGroup.Item>Dapibus ac facilisis in</ListGroup.Item>
    <ListGroup.Item>Morbi leo risus</ListGroup.Item>
    <ListGroup.Item>Porta ac consectetur ac</ListGroup.Item>
    <ListGroup.Item>Vestibulum at eros</ListGroup.Item>
  </ListGroup>
</ComponentPreview>

## Active & Disabled Items

Set the `active` prop to indicate the list groups current active selection.
Set the disabled prop to prevent actions on a `<List.Item>`. For elements that aren't naturally disable-able (like anchors) onClick handlers are added that call preventDefault to mimick disabled behavior.

<ComponentPreview>
  <Row>
    <Col>
      <h4>Active</h4>
      <ListGroup as="ul">
        <ListGroup.Item as="li" active>
          Cras justo odio
        </ListGroup.Item>
        <ListGroup.Item as="li">Dapibus ac facilisis in</ListGroup.Item>
        <ListGroup.Item as="li" active>
          Morbi leo risus

        <ListGroup.Item as="li">Porta ac consectetur ac</ListGroup.Item>

    </Col>

## Tabbed List groups

You can also use the `Tab` components to create ARIA compliant tabbable interfaces with the `<ListGroup>` component. Swap out the `<Nav>` component for the list group and you are good to go.

<ComponentPreview>
  <Tab.Container id="list-group-tabs-example" defaultActiveKey="first">
    <Row>
      <Col sm={4}>
        <ListGroup>
          <ListGroup.Item eventKey="first" action>
            Tab 1
          </ListGroup.Item>
          <ListGroup.Item eventKey="second" action>
            Tab 2
          </ListGroup.Item>
        </ListGroup>
      </Col>
      <Col sm={8}>
        <Tab.Content>
          <Tab.Pane eventKey="first">
            <p>
              Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce
              iaculis nisl non elementum tristique. Donec eu ex facilisis,
              molestie nisl non, vestibulum est. Curabitur gravida quis lectus
              at vulputate.
            </p>
          </Tab.Pane>
          <Tab.Pane eventKey="second">
            <p>
              Interdum et malesuada fames ac ante ipsum primis in faucibus. Cras
              at velit vitae elit mattis dictum quis ac nulla. Morbi et ante
              lorem. Maecenas condimentum ante ex, sed suscipit leo posuere eu.
              Proin vel est iaculis, hendrerit augue ac, tempus est.
            </p>{' '}
          </Tab.Pane>
        </Tab.Content>
      </Col>
    </Row>
  </Tab.Container>
</ComponentPreview>
