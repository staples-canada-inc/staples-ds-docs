# Avatars

Avatars are typically profile images that either contain a placeholder, initials, or a custom image that the user can upload.

## Avatar with Image

Avatars with an image use the `<img>` tag and include the `.avatar-img` class. This class uses css to manipulate an image into an avatar. It can also be used for avatar placeholder icons. 

<ComponentPreview>
    <img src="https://source.unsplash.com/216x216?face" alt="..." class="avatar-img"/>
</ComponentPreview>

```html

<img src="#" alt="..." class="avatar-img"/>

```

### Avatar Sizes

There are different sizes available for avatars {sm|lg|xl} in addition to the default size. 

<ComponentPreview>
    <img src="https://source.unsplash.com/216x216?face" alt="..." class="mr-3 avatar-img-sm"/>
    <img src="https://source.unsplash.com/216x216?face" alt="..." class="mr-3 avatar-img"/>
    <img src="https://source.unsplash.com/216x216?face" alt="..." class="mr-3 avatar-img-lg"/>
    <img src="https://source.unsplash.com/216x216?face" alt="..." class="mr-3 avatar-img-xl"/>
</ComponentPreview>

```html
<img src="#" alt="..." class="avatar-img-sm"/>
<img src="#" alt="..." class="avatar-img"/>
<img src="#" alt="..." class="avatar-img-lg"/>
<img src="#" alt="..." class="avatar-img-xl"/>
```

## Avatar With User Initials

For creating avatars with the user's initials, use the `.avatar` class which creates the size of the image and background. Before inputting the initials, use the `span` tag and then use our custom `initials` class to properly apply the initials in the avatar. 

Once again, We also have different sizes available for these avatars {sm|lg|xl} in addition to the default size (MD in this case).

<ComponentPreview>
<div class="mb-3 avatar-sm">
  <span class="initials-sm">SM</span>
</div>
<div class="mb-3 avatar">
  <span class="initials">MD</span>
</div>
<div class="mb-3 avatar-lg">
  <span class="initials-lg">LG</span>
</div>
<div class="mb-3 avatar-xl">
  <span class="initials-xl">XL</span>
</div>
</ComponentPreview>

```html
<div class="avatar-sm">
  <span class="initials-sm">SM</span>
</div>
    
<div class="avatar">
  <span class="initials">MD</span>
</div>

<div class="avatar-lg">
  <span class="initials-lg">LG</span>
</div>

<div class="avatar-xl">
  <span class="initials-xl">XL</span>
</div>
```

## Usage

Avatars are usually used as profile icons and in toasters when getting notifications. They can be customized to the user and serve as identification.
